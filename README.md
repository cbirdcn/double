# Double（梅开二度）

## 介绍

足球俱乐部养成游戏，原型为[hattrick](https://m.hattrick.org/)
- rpc
- http


## By Bilibili Kratos

前期准备见[kratos](https://github.com/go-kratos/kratos)

```
go get -u github.com/go-kratos/kratos/cmd/kratos/v2@latest
```

## Docker

部署golang容器，开放8000做http端口，9000做rpc端口

## RUN

kratos run
